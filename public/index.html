    VS Code Demo

    This web application demonstrates a the interaction with a Fabric network using VS Code exported components

    Learn more »

    The package "fabric-sdk-node" is used to connect and transact with an existing Hyperledger Fabric network.

    Learn more »

    This demo assumes that a network exists, and has been connected to via VS Code, from which a connection profile and wallet have been exported. Please note that if you tear down your network, you will need to re-export the wallet as crypto-material is regenerated each time the network is created.

    Submit Transaction

    Submitting a transaction will result on the contract method being run on Hyperledger Fabric Peers as required by the endorsement policy and appended to the ledger by the ordering service.

    Evaluate Transaction

    Evaluating a transaction will not include interaction with the ordering service, and consequently will not result in appending to the leger.
